Memory Game

Hello and welcome to my memory game. Help train and develop your own skills and memory with this fun simple game.

How to Start

To start the game use the start button and then this button will disappear and the four coloured panels will appear and then the counter will increase to 1 and one panel will flash once it the panel is the original colour click the panel and if correct the counter will increase by 1 and that same panel will flash again with an additional one after it untill the counter is at 20 which is the last round

What is one life

if one life is checked that means if a colour is incorrectly clicked then the panels will flash white and round will be reset to 1

What is Super Speed

if Super Speed is checked that will reduce the speed of the flashes and the players flashes by half

What is Unlimited Rounds

if Unlimited Rounds is checked that will increase the final round from 20 to unlimited

all buttons can be turned on and off at any point and also can be put all on together
